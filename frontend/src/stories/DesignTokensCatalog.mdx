<Subtitle>基本トークン定義（Style Dictionary）</Subtitle>

このページは、`frontend/tokens` で定義したデザイントークンの設計意図と利用ルールをまとめたカタログです。  
出力CSSは `src/styles/tokens.css` に生成され、アプリ本体とStorybookで共通利用されます。

## 設計方針

- `core`: 生の値（色、余白、タイポなど）
- `semantic`: 役割ベースの値（背景、テキスト、ボタン、入力）
- 原則としてコンポーネント実装では `semantic` を利用し、`core` を直接参照しない

## 使い方（CSS）

```css
.button-primary {
  height: var(--semantic-component-button-height);
  padding-inline: var(--semantic-component-button-padding-x);
  border-radius: var(--semantic-component-button-radius);
  background: var(--semantic-component-button-bg);
  color: var(--semantic-component-button-text);
}

.input-default {
  height: var(--semantic-component-input-height);
  border: 1px solid var(--semantic-component-input-border);
  color: var(--semantic-component-input-text);
}
```
